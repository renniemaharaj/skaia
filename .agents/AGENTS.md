# Rules
- **NEVER** automatically run `git commit` for backend changes. Always ask for user confirmation first or let the user do it.
\n- **NEVER** use scripts (like Node/Python/bash scripts) to edit or refactor files. Always use the built-in file editing tools manually.
- **NEVER** automatically run `git commit` at all unless explicitly asked or permitted. Always ask for user confirmation first or let the user do it.
