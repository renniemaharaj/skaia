# .todo/.routines

This directory contains routine definitions for project maintenance and automation. Each file describes a routine that models can perform, including responsibilities, references, expectations, goals, last run info, remarks, and next steps.

## Routine File Structure

- **Responsibilities**: What the routine covers
- **References**: Where to look for context
- **What's Expected**: Deliverables or checks
- **Goal**: The main objective
- **Last Ran**: When the routine was last performed
- **Last Run Remarks**: Notes from the last execution
- **Next Steps**: What to do next
